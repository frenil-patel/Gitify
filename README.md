# Git Drag-n-Drop (MVP)

MVP extension to display local Git branches in a React + Material UI webview. Later, enable drag-and-drop Git operations.

## Commands
- `Git DnD: Show Branches` â€” opens the branches webview.

## Development
- Build extension: `npm run build`
- Build webview: `npm run build:webview`
- Launch Extension: Press F5 in VS Code

Requires an open workspace with a Git repository.
